# Shadowrun Items Importer

Got a stash of Shadowrun 6e qualities, gear, or custom content sitting in PDFs, notes, or rulebooks? Tired of manually recreating every single item in Foundry?

Shadowrun Items Importer has you covered.

Simply copy and paste an item directly into the import dialog, select the type, and Voilà! — your item is instantly created and ready to use or customize in your world.

Built specifically for Shadowrun 6th Edition, the importer intelligently parses typical rulebook formatting, handling messy copy-paste text, bullet lists, and multiline descriptions with ease. No strict formatting required — just paste and go - some finger crossed will help anyway.

Spend less time typing, and more time running the shadows.

## Supported system target

The scaffold is aimed at the active `shadowrun6-eden` Foundry system.

## Object parsed

**Qualities**: as from Core and Compendium format
**Matamagic**: as for Core
**Spells**: as for Core
**Gear - Chemicals - Toxin**: as for Core
